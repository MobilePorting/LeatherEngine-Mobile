class StorageUtil

package mobile.utilities

A storage utility class for mobile platforms. Provides methods for handling storage directories, creating directories, saving content, and requesting permissions for android.

Static methods

@:value({ force : false })staticgetStorageDirectory(force:Bool = false):String

Gets the storage directory based on the platform.

Returns:

The path to the storage directory.

@:value({ alert : true })staticsaveContent(fileName:String, fileData:String, alert:Bool = true):Void

Saves content to a file in the saves directory.

Parameters:

fileName

The name of the file.

fileExtension

The extension of the file. Defaults to '.json'.

fileData

The content to save in the file. Defaults to a placeholder string.
